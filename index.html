class VELA_CLARA_Scope_Control.scopeNumObject¶

This struct contains data for the four channels on the scope - e.g. p1 contains the P1 value from the scope. This will only contain real-time data if scope number data is being submitted to EPICS - you will need to check this on the scope. p1Vec contains a vector of values after monitorNumsForNShots, and numData contains a vector of vectors, with data from all four channels on the scope. Timestamps can also be accessed.
